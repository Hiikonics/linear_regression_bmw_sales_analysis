# linear_regression_bmw_sales_analysis
This repo features a Jupyter Notebook for predicting BMW `units_sold` using Linear Regression (Scikit-learn).  Key features: - Data prep &amp; EDA on BMW global sales data. - Model training with `marketing_spend_usd`, `dealership_count`, `price_usd`, `competition_index`. - Achieved R-squared: 0.76. - Visualizations &amp; coefficient analysis included.
